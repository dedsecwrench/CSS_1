What is css ?


  1. CSS stands for cascading stylesheet

  2. CSS describes how HTML elements are to be displayed on screen, paper or in other media.

  3. Css is not neccesary CSS saves a lot of work. it can control the layout of mutiple web pages all at once

  4. External shtylesheets are stored in css files.

Css can be added to html documents in 3 ways :


  • Inline - By using style attribute inside HTML element.
  • Internal - By using a style element in head section.
  • External - By using a link element to link to an external CSS file.
  • The most common way to add CSS,is to keep the styles in external css files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID,on the other hand applies style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by javascript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set od styles can be applied to the elements.

There is a wide range of selectors available, Two of the most common used are class and ID. both are used to target elements to which a style should be applied.
